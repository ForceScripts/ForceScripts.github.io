Play with Keyboard

Playing With A Keyboard Is The Easiest To Set Up But If You Want Try Using A Xbox Controller.
Arrow Keys = Move
X Key = Jump, Read Dialouge, Select Option
C Key = Grab, Throw, Punch
W, A, S, D = Camera Controls
Spacebar = Crouch, Crawl, Ground Pound
Enter = Pause, Start

Play with Physical Xbox Controller

To Play With A Xbox Controller, Just Plug It And Play.
